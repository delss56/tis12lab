№1:
Название:	Авторизация с некорректными учетными данными.
Предусловие:	Пользователь не авторизирован в системе.
Шаги:	
1.	Переходим на сайт Steam на страницу авторизации;
2.	В поле «Войти, используя имя аккаунта» вводим 'example123';
3.	В поле «Пароль» вводим '321elpmaxe';
4.	Нажимаем на кнопку «Вход».
Ожидаемый результат	Пользователь находится на странице авторизации. На странице отображается сообщение «Неверный логин или пароль, попробуйте заново.».

№2:
Название:	Поиск конкретной игры.
Предусловие:	Пользователь не авторизирован в системе.
Шаги:	
1.	В поле «Поиск» вводим ‘terraria’;
2.	Нажимаем на первый результат поиска.
Ожидаемый результат	Пользователь находится на странице определенной игры.

№3:
Название:	Поиск по жанрам игр
Предусловие:	Пользователь не авторизирован в системе.
Шаги:	
1.	В поле «Поиск» вводим ‘Головоломки’;
2.	Нажимаем на иконку/кнопку «Поисковой лупы»
3.	Сортируем по обзорам.
4.	Нажимаем на первый результат
Ожидаемый результат	Пользователь находится на странице игры со списка.
